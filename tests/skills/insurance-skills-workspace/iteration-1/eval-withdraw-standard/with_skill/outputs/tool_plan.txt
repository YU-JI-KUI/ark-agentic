→ 结果: 见题目中的 JSON（POL002 年金险 + POL001 终身寿险）

### Step 3: 方案组装逻辑（推导过程）

目标金额：50,000 元
用途：education → 优先零成本渠道

【渠道分析】
POL002 金瑞人生年金险（annuity）:
  - survival_fund_amt: 12,000（零成本）
  - bonus_amt: 5,200（零成本）
  - refund_amt: 160,000（部分领取，非 whole_life，手续费 1%）
  - loan_amt: 0（无贷款）

POL001 安心守护终身寿险（whole_life）:
  - survival_fund_amt: 0
  - bonus_amt: 0
  - refund_amt: 33,600（退保，whole_life，保障终止）
  - loan_amt: 42,000（贷款，年利率 5%）

【⭐ 推荐方案组合】（零成本优先 + 部分领取补足）
  POL002 生存金 12,000 + 红利 5,200 + 部分领取 32,800 = 50,000 元
  手续费：32,800 × 1% = 328 元（仅对部分领取收取）
  Impact：年金保额略降，生存保障继续有效

【备选方案组合】（保单贷款 + 零成本，保障完整）
  POL002 生存金 12,000 + 红利 5,200（零成本）
  POL001 贷款 32,800（≤ max 42,000，年利率 5%，约 1,640 元/年）
  合计 = 50,000 元，无手续费，两张保单保障均完整保留

（排除退保：whole_life 退保会终止 POL001 所有保障，且 33,600 不足 50,000；不到万不得已不用）

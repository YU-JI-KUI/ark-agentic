# 项目概述

> 这个文件用于记录项目的概述信息，帮助 Claude 快速了解项目。

## 项目名称

**ark-agentic** - 轻量级 ReAct Agent 框架

## 项目描述

一个功能完整的 Python 智能体框架，支持：
- **ReAct 模式**: 推理-行动循环，支持并行工具调用
- **多 LLM 支持**: DeepSeek、PA 内部模型 (JT/SX 系列)、OpenAI 兼容端点
- **技能系统**: Markdown 格式可复用指令集，支持 full/dynamic/semantic 三种加载模式
- **会话管理**: JSONL 持久化 + 智能上下文压缩（LLM 摘要）+ Session State 状态管理
- **向量记忆**: FAISS + Sentence-Transformers 语义搜索 + 混合检索（RRF）
- **AG-UI 流式协议**: 完整的 17 种事件类型，支持 4 种输出格式（agui/internal/enterprise/alone）
- **A2UI 组件**: 支持富交互前端组件渲染（卡片、按钮、表单等）
- **输出验证**: 自动检测 LLM 输出与工具结果的数值一致性，防止幻觉
- **CLI 工具**: `ark-agentic` 命令行工具，支持项目脚手架和智能体生成
- **FastAPI 服务**: 生产就绪的 HTTP API，支持多协议 SSE 流式输出

## 技术栈

- **语言**: Python 3.12+
- **核心框架**: LangChain (LLM 抽象)
- **Web 框架**: FastAPI (API 服务)
- **向量存储**: FAISS
- **嵌入模型**: Sentence-Transformers
- **依赖管理**: uv
- **持久化**: JSONL
- **流式协议**: SSE (Server-Sent Events)

## 开发说明

### 环境变量

| 变量 | 说明 | 默认值 |
|------|------|--------|
| `LLM_PROVIDER` | LLM 提供商 (pa/deepseek/openai/mock) | `deepseek` |
| `PA_MODEL` | PA 模型 (PA-JT-80B/PA-SX-80B/PA-SX-235B) | - |
| `DEEPSEEK_API_KEY` | DeepSeek / OpenAI 兼容端点 API Key | - |
| `DEFAULT_TEMPERATURE` | LLM 温度 | `0.7` |
| `API_HOST` | API 监听地址 | `0.0.0.0` |
| `API_PORT` | API 端口 | `8080` |
| `SESSIONS_DIR` | 会话存储目录 | `/data/sessions` |
| `MEMORY_DIR` | Memory 数据目录 | `/data/memory` |
| `SECURITIES_SERVICE_MOCK` | 证券服务 Mock 模式 | `true` |
| `SECURITIES_ACCOUNT_TYPE` | 证券账户类型 (normal/margin) | `normal` |

### 快速开始

```python
from ark_agentic.core.runner import AgentRunner
from ark_agentic.core.session import SessionManager
from ark_agentic.core.tools.registry import ToolRegistry
from ark_agentic.core.llm import create_chat_model

# 使用保险智能体
from ark_agentic.agents.insurance.tools import create_insurance_tools

# 或使用证券智能体
from ark_agentic.agents.securities import create_securities_agent

llm = create_chat_model("deepseek-chat", api_key="sk-xxx")

# 方式一：手动创建
tool_registry = ToolRegistry()
tool_registry.register_all(create_insurance_tools())

agent = AgentRunner(
    llm=llm,
    tool_registry=tool_registry,
    session_manager=SessionManager(),
)

# 方式二：使用工厂函数
agent = create_securities_agent(llm=llm)

session_id = await agent.create_session()
result = await agent.run(session_id, "我想取点钱")
```

### 核心概念

**工具定义**: 继承 `AgentTool` 基类，支持返回 JSON/TEXT/IMAGE/A2UI/ERROR 五种结果类型

**技能系统**: Markdown 格式可复用指令集，支持 full/dynamic/semantic 三种加载模式

**会话压缩**: 使用 LLM 自动总结历史消息，保持上下文窗口稳定

**Session State**: 跨工具调用的状态管理，通过 `metadata.state_delta` 写入

**向量记忆**: FAISS + Sentence-Transformers 语义搜索，支持混合检索（RRF）

### 智能体示例

#### 保险智能体 (insurance)
- 保单查询
- 客户信息查询
- 取款业务处理
- 规则引擎验证

#### 证券智能体 (securities)

- 现金资产查询
- ETF/港股通/基金持仓查询
- 标的详情查询
- A2UI 卡片渲染

### AG-UI 流式协议

支持 4 种输出协议：
- `agui`: AG-UI 原生事件（17 种事件类型）
- `internal`: 旧版 response.* 格式（向后兼容）
- `enterprise`: 企业 AGUI 信封格式
- `alone`: ALONE 协议（sa_* 事件）

# 收益查询技能

## 目标

帮助用户了解投资收益情况，包括今日收益、累计收益、各类资产收益对比等。

## 触发条件

当用户询问以下内容时触发此技能：
- "今天赚了多少"
- "总收益多少"
- "哪个资产收益最好"
- "收益率排名"
- 其他类似的收益查询

## 执行步骤

1. **确定查询范围**：
   - 总收益 → 调用 `account_overview(return_json_card=True/False)`
   - 单类资产收益 → 调用对应工具
   - 具体标的收益 → 调用 `security_detail(..., return_json_card=True/False)`

2. **计算收益指标**：
   - 今日收益
   - 累计收益
   - 收益率
   - 年化收益率（如适用）

3. **收益分析**：
   - 识别收益来源
   - 对比市场表现
   - 提供优化建议

## 输出格式

### 场景 1：总收益查询（返回 JSON 模板）

**触发条件**：
- "今天赚了多少"、"总收益"等简单查询

**返回格式**：
直接输出工具返回的 JSON 数据：

```text
{ "template_type": "profit_summary_card", "data": { ... } }
```

### 场景 2：收益分析（返回 Markdown）

**触发条件**：
- "为什么今天亏损"、"收益来源分析"

**返回格式**：
```markdown
## 收益分析

### 今日收益

- 📊 **今日盈亏**：+¥15,000 (+1.2%)
- 🎯 **累计收益**：+¥250,000 (+25.0%)

### 收益来源

| 资产类别 | 今日收益 | 贡献度 |
|---------|---------|--------|
| ETF | +¥300 | 2% |
| 港股通 | +¥2,500 | 16.7% |
| 基金理财 | +¥50 | 0.3% |
| 其他 | +¥12,150 | 81% |

### 分析

今日收益主要来自港股通板块，腾讯控股上涨带动...
```

## 收益排名

展示各持仓的收益率排名：

```markdown
## 收益率排名 TOP 3

1. 🥇 **创业板ETF**：+6.82%
2. 🥈 **沪深300ETF**：+6.67%
3. 🥉 **招商白酒基金**：+12.5%
```

## 亏损提示

如果有亏损，需要特别提示：

```markdown
⚠️ **亏损提示**

以下持仓当前处于亏损状态：
- 某某股票：-5.2%（建议关注）
```

## 注意事项

1. **数据准确性**：确保收益计算准确
2. **时间范围**：明确收益的时间范围（今日/累计）
3. **风险提示**：过往收益不代表未来表现

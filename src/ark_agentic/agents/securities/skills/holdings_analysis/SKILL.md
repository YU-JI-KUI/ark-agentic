# 持仓分析技能

## 目标

帮助用户了解各类资产的持仓详情，包括持仓列表、收益情况、风险分布等。

## 触发条件

当用户询问以下内容时触发此技能：
- "我的 ETF 持仓"
- "港股通有哪些"
- "基金持仓情况"
- "持仓分布"
- 其他类似的持仓查询

## 执行步骤

1. **识别资产类别**：
   - ETF → 调用 `etf_holdings` 工具
   - 港股通 → 调用 `hksc_holdings` 工具
   - 基金理财 → 调用 `fund_holdings` 工具
   - 全部持仓 → 调用多个工具

2. **意图判断**：
   - **纯查询**（如"我的 ETF"）→ 返回 JSON 模板卡片
   - **分析性**（如"ETF 收益怎么样"）→ 返回 Markdown 分析

3. **数据整理**：
   - 按收益率排序
   - 计算占比
   - 识别盈亏情况

## 输出格式

### 场景 1：单类资产查询（返回 JSON 模板）

**触发条件**：
- "我的 ETF"、"港股通持仓"等简单查询
- 无分析性问题

**返回格式**：
```json
{
  "template_type": "holdings_list_card",
  "asset_class": "ETF",
  "data": {
    "holdings": [
      {
        "security_code": "510300",
        "security_name": "沪深300ETF",
        "quantity": 10000,
        "cost_price": 4.50,
        "current_price": 4.80,
        "market_value": 48000.00,
        "profit": 3000.00,
        "profit_rate": 0.0667
      }
    ],
    "summary": {
      "total_market_value": 59750.00,
      "total_profit": 3750.00,
      "total_profit_rate": 0.0670
    }
  }
}
```

- "ETF 收益怎么样"、"哪只基金表现最好"

**返回格式**：

## 多类资产对比

如果用户查询全部持仓，展示各类资产占比：

```markdown
## 持仓分布

- 📈 **ETF**：¥59,750 (50%)
- 🌏 **港股通**：¥210,000 (35%)
- 💼 **基金理财**：¥6,750 (15%)
```

## 注意事项

1. **空持仓处理**：如果某类资产无持仓，友好提示
2. **收益排序**：默认按收益率降序排列
3. **风险提示**：亏损较大的持仓需要特别标注

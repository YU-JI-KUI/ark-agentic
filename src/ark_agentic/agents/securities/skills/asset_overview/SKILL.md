# 资产总览技能

## 目标

帮助用户快速了解账户的整体资产状况，包括总资产、现金、持仓市值、收益等关键指标。

## 触发条件

当用户询问以下内容时触发此技能：
- "查看我的资产"
- "账户总览"
- "我的资产情况"
- "总资产多少"
- 其他类似的账户整体查询

## 执行步骤

1. **调用工具**：使用 `account_overview` 工具获取账户总资产数据
2. **判断账户类型**：
   - 普通账户：展示基本资产信息
   - 两融账户：额外展示维持担保比率、风险等级等
3. **意图判断**：
   - 如果用户**仅查看总览**（无具体问题）→ 返回 JSON 模板卡片
   - 如果用户有**具体问题**（如"为什么亏损"）→ 返回 Markdown 文本分析

## 输出格式

### 场景 1：纯总览查询（返回 JSON 模板）

**触发条件**：
- 用户输入简单，如"查看资产"、"账户总览"
- 没有额外的分析性问题

**返回格式**：
```json
{
  "template_type": "account_overview_card",
  "data": {
    "total_assets": 1250000.00,
    "cash_balance": 50000.00,
    "stock_market_value": 1200000.00,
    "today_profit": 15000.00,
    "total_profit": 250000.00,
    "profit_rate": 0.25,
    "account_type": "normal"
  }
}
```

### 场景 2：分析性查询（返回 Markdown）

**触发条件**：
- 用户询问具体问题，如"为什么今天亏损"、"资产配置合理吗"

**返回格式**：
```markdown
## 您的资产总览

- 💰 **总资产**：¥1,250,000.00
- 💵 **可用资金**：¥50,000.00
- 📈 **持仓市值**：¥1,200,000.00
- 📊 **今日收益**：+¥15,000.00 (+1.2%)
- 🎯 **累计收益**：+¥250,000.00 (+25.0%)

### 分析

今日收益为正，主要得益于...
```

## 两融账户特殊处理

如果是两融账户，需要额外展示：

```markdown
**两融账户风险指标**：
- 🛡️ **维持担保比率**：280%
- ⚠️ **风险等级**：低风险
- 💰 **维持保证金**：¥850,000
- 💵 **可用保证金**：¥1,650,000
```

## 注意事项

1. **数据时效性**：提醒用户数据更新时间
2. **风险提示**：两融账户需要特别提示风险等级
3. **简洁性**：总览应简洁明了，避免过多细节
